# 학습 노트

이 폴더는 Verilog/Vivado 실습 중 배운 내용을 짧게 정리하는 공간입니다.

## 정리 기준

- 실습 목표
- 주요 모듈과 입출력
- 동작 원리
- 구현하면서 헷갈렸던 점
- 보드에서 확인한 결과
